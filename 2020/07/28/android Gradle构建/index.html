  1. Initialization

    阶段主要目的是初始化构建，又分为两个子过程

    1. Init Script => Init script 会读取全局脚本

 2. Setting Script. =>它初始化了一次构建所参与的所有模块
    1. Configuration

      执行 build.gradle(.kts) 中的语句, 根据脚本代码创建对应的 task, 最终根据所有 task 生成对应的依赖图.

  2. Execution

    进行编译和打包动作
